# AI Research Assistant - Multi Agent Pipeline

Sistem multi-agent berbasis AI yang secara otomatis mencari, merangkum, 
menganalisis, dan membuat laporan riset dari paper ilmiah.

## Fitur
- Otomatis mencari paper dari Arxiv
- Merangkum temuan utama tiap paper
- Membandingkan metodologi antar paper
- Menghasilkan laporan riset terstruktur

## Tech Stack
- Python
- Groq API (LLaMA 3.3 70B)
- LangChain
- Google Colab

## Cara Pakai
1. Buka notebook di Google Colab
2. Masukkan Groq API key di Secrets
3. Jalankan semua cell
4. Ganti query sesuai topik riset kamu
